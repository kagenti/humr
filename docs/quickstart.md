# Quickstart

Log in, add your API key, create an agent, and start chatting — all in under two minutes.

## 1. Log in

Open the Humr URL you've been given and log in with your credentials.

## 2. Add your API key

Go to **Providers**, pick your provider (e.g. Anthropic), and paste your API key or OAuth token. This only needs to happen once — see [Add Credentials](user-guide/credentials.md) for details.

## 3. Create an agent

Click **Add Agent**, pick a template (e.g. Claude Code), and give it a name. The agent starts in ~20 seconds.

## 4. Start chatting

Click on your agent and send a message. That's it — your agent has a full Linux environment, persistent storage, and access to the credentials you just configured.
