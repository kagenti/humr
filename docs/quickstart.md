# Quickstart

*Already have access to a hosted Humr instance? This is for you.* Add your API key, create an agent, and start chatting — all in under two minutes.

If you want to run Humr yourself, head to [Self-Hosting](self-hosting/get-started.md) instead.

## 1. Add your API key

Log in to your Humr instance, go to **Providers**, pick your provider (e.g. Anthropic), and paste your API key or OAuth token. This only needs to happen once — see [Credentials](user-guide/credentials.md) for details.

## 2. Create an agent

Click **Add Agent**, pick a template (e.g. Claude Code), and give it a name. The agent starts in ~20 seconds.

## 3. Start chatting

Click on your agent and send a message. That's it — your agent has a full Linux environment, persistent storage, and access to the credentials you just configured.
